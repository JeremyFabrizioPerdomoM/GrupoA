Centro de Ayuda

Soporte y orientación para el uso del sitio web institucional

Bienvenido al centro de ayuda

Esta sección está diseñada para orientarte en el uso del sitio web del instituto y resolver dudas generales sobre navegación, acceso a información y funcionamiento de la plataforma.

Cómo navegar en el sitio

  • Utiliza el menú principal para acceder a las distintas secciones del sitio.
  • Desplázate hacia abajo en cada página para ver todo el contenido disponible.
  • Explora cada sección para encontrar información detallada sobre el instituto.

Problemas comunes

El sitio no carga correctamente

Intenta actualizar la página o verificar tu conexión a internet. Si el problema persiste, prueba desde otro navegador.

Los enlaces no funcionan

Asegúrate de que estás utilizando la versión más reciente del sitio. Algunos enlaces pueden cambiar con actualizaciones.

Contenido no visible

Verifica que tu navegador tenga habilitado JavaScript y que no existan bloqueadores que afecten la visualización.

Recomendaciones de uso

  • Usa un navegador actualizado (Chrome, Firefox o Edge).
  • Evita el uso de redes lentas para una mejor experiencia.
  • Mantén tu dispositivo actualizado para evitar errores de compatibilidad.

Accesibilidad

Este sitio está diseñado para ser accesible en computadoras, tablets y teléfonos móviles. Puedes ampliar el texto usando las opciones de tu navegador si lo necesitas.

Nota importante

Este centro de ayuda no reemplaza los canales oficiales de comunicación del instituto, sino que sirve como guía rápida para el uso del sitio web.
